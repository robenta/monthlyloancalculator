Monthly Loan Calculator

A simple web-based loan calculator built using HTML, CSS, and JavaScript. 
This project includes dynamic calculations and manipulating the webpage.

Features:

Calculate monthly payments based on loan amount, interest rate, and loan term

Instant results without page reload

Basic CSS styling for a clean, user-friendly interface

Responsive layout for desktop and mobile

Purpose:
This project was created to practice JavaScript calculations, handling user input, and updating the webpage dynamically based on calculations.
